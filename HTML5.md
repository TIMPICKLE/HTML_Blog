## 常用的表章节的标签有哪些，分别是什么意思（h1~h6、section、article、main、aside 等等）
* h1~h6 标题
* section 章节
* article 文章
* p 段落
* header 头部
* footer 脚部
* main 主要内容
* aside 旁枝内容
* div 划分

## 全局属性有哪些
* class
* contenteditable
* hidden
* id
* style
* tabindex
* title

## 常用的内容标签有哪些，分别是什么意思（a、strong、em、code、pre 等等）
* <ol><li> order list
* <ul><li>  unorder list
* <dl><dt><dd> description list/term/data
* <pre> 保留回车，空格等操作
* <code> 代码块
* <hr> 分割线
* <br> 换行
* <a> 超链接
* <em> 强调（语气）
* <quote> 引用（行内）
* <blockquote> 引用（块级）
